## ğŸš€ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/ieee-isetb/CIS.git
   cd CIS
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173/CIS/`

4. **Build for production**
   ```bash
   npm run build
   ```
